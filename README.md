# stratford.dev

Marketing site for **stratford.dev** — promoting a startup culture in the West Midlands. Networking, helping founders, and using AI in Stratford-upon-Avon.

Static HTML/CSS, no build step.

## Run locally

```bash
cd website
python3 -m http.server 8093 --bind 0.0.0.0
```

Then visit <http://localhost:8093>.

## Deploy

Hosted on a Hetzner VM via [Dokploy](https://dokploy.com/), deployed from this repo. DNS via Cloudflare.
